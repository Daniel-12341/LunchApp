  <action>
1. Install framer-motion: `npm install framer-motion`

2. Create `src/components/PageTransition.tsx` ('use client'):
   - Import `motion`, `AnimatePresence` from 'framer-motion'
   - Import `usePathname` from 'next/navigation'
   - Use `usePathname()` as the AnimatePresence key (simpler than FrozenRouter, avoids internal import breakage per research pitfall 1)
   - AnimatePresence with mode="wait" initial={false}
   - motion.div with: initial={{ opacity: 0, y: 8 }}, animate={{ opacity: 1, y: 0 }}, exit={{ opacity: 0, y: -8 }}, transition={{ duration: 0.2, ease: 'easeInOut' }}
   - Wrap children in the motion.div

3. Update `src/app/layout.tsx`:
   - Remove Geist and Geist_Mono imports
   - Import `Pacifico` and `Fredoka` from 'next/font/google' (Note: "Fredoka One" was merged into "Fredoka" in Google Fonts — use `Fredoka` with weight '700' for the bold rounded look. If Fredoka import fails, fall back to importing as `Fredoka` with weight '400'. Verify the exact export name works at import time.)
   - Configure: `const pacifico = Pacifico({ weight: '400', subsets: ['latin'], variable: '--font-pacifico', display: 'swap' })`
   - Configure: `const fredoka = Fredoka({ weight: '400', subsets: ['latin'], variable: '--font-fredoka', display: 'swap' })` (try weight '400' first since Fredoka is variable-weight)
   - Import PageTransition from '@/components/PageTransition'
   - Update body className to `${pacifico.variable} ${fredoka.variable} antialiased`
   - Wrap {children} with `<PageTransition>{children}</PageTransition>`

4. Update `src/app/globals.css`:
   - In @theme inline block: replace `--font-sans: var(--font-geist-sans)` with `--font-sans: Arial, Helvetica, sans-serif` (body stays clean sans-serif per user decision)
   - Remove `--font-mono: var(--font-geist-mono)` line
   - Add after @theme inline block, a `@layer utilities` block with:
     ```css
     .font-pacifico { font-family: var(--font-pacifico), cursive; }
     .font-fredoka { font-family: var(--font-fredoka), sans-serif; }
     ```
