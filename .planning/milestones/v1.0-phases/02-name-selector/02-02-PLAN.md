**Step 2: Replace src/app/home/page.tsx**

Replace the current placeholder with a Server Component that:
- Runs auth check (createClient + getUser, redirect if no user)
- Uses `next/dynamic` to import NameSelectorPage with `ssr: false`
- Renders the dynamically imported component
- Shows a loading state (full-screen emerald pulse animation) while loading

```typescript
import { redirect } from 'next/navigation'
import { createClient } from '@/utils/supabase/server'
import dynamic from 'next/dynamic'

const NameSelectorPage = dynamic(() => import('@/components/NameSelectorPage'), {
  ssr: false,
  loading: () => <div className="w-full h-screen bg-emerald-50 animate-pulse" />,
})

export default async function HomePage() {
  const supabase = await createClient()
  const { data: { user } } = await supabase.auth.getUser()
  if (!user) redirect('/')

  return <NameSelectorPage />
}
```

  - Animated entrance: transition from opacity-0 translate-y-4 to opacity-100 translate-y-0 (CSS transition, 300ms)

**Step 4: Create src/components/MapComponent.tsx**

Client component ('use client') containing:

1. Import leaflet CSS at the top: `import 'leaflet/dist/leaflet.css'`
2. Import L from 'leaflet', MapContainer, TileLayer, Marker from 'react-leaflet', useMap from 'react-leaflet'

3. `createEmojiIcon` function: Creates an `L.divIcon` for each person
   - HTML: `<div class="emoji-marker {selected|dimmed} {bounce}">{emoji}<span class="marker-location">{locationName}</span></div>`
   - The emoji is large (text-2xl), the location name is small text below
   - className: '' (clear leaflet defaults)
   - iconSize: [60, 50], iconAnchor: [30, 45]

4. `FlyController` component (rendered inside MapContainer):
   - Props: `target: { lat, lng } | null`, `zoom: number`, `onAnimationEnd: () => void`
   - Uses `useMap()` to get map instance
   - useEffect watching `target`: calls `map.flyTo([target.lat, target.lng], zoom, { animate: true, duration: 2.5 })`
   - Registers one-time `moveend` listener that calls `onAnimationEnd`
   - Cleanup: removes listener on unmount / re-trigger
   - Guard: if target is null, do nothing

5. MapComponent main:
   - Props: `people: Person[]`, `selectedPerson: Person | null`, `onAnimationEnd: () => void`
   - Track `bouncingName: string | null` state — set to selectedPerson.name in onAnimationEnd, cleared after 2 seconds via setTimeout
   - MapContainer: center=[-33.9, 18.8] (Western Cape overview), zoom=9, style={{ height: '100vh', width: '100%' }}, scrollWheelZoom=true, zoomControl=true
   - TileLayer: CartoDB Voyager tiles `https://{s}.basemaps.cartocdn.com/rastertiles/voyager/{z}/{x}/{y}{r}.png` with proper attribution
   - Map over `people` array rendering a Marker for each:
     - position: [person.lat, person.lng]
     - icon: createEmojiIcon with:
       - isSelected: person.name === selectedPerson?.name
       - isBouncing: person.name === bouncingName
     - key: person.name
   - FlyController: target = selectedPerson ? { lat, lng } : null, zoom = 14, onAnimationEnd wraps parent callback + sets bouncingName

**Critical implementation notes (from research):**
- NEVER import leaflet in a Server Component — only in 'use client' files
- MapContainer MUST have `style={{ height: '100vh', width: '100%' }}` — Tailwind classes alone won't work for Leaflet
- Only attach `moveend` listener AFTER a flyTo is triggered (not on mount) to prevent Continue appearing on page load
- Use `L.divIcon` for all markers — do NOT use default Leaflet marker icons (they 404 in Next.js)
- Clean up moveend listeners in useEffect return to prevent accumulation
